🌦️ Weather App (JavaScript)
This is a simple weather application built with HTML, CSS, and JavaScript.
It uses the OpenWeatherMap API to fetch real-time weather data for any city.

- Features
Search weather by city name
Displays temperature, humidity, and wind speed
Dynamic weather icons (clouds, rain, snow, etc.)
Error handling for invalid cities (404)

- How it works
The app sends a request to the OpenWeatherMap API using the city name entered by the user.
It then displays:
City name
Temperature (°C)
Humidity (%)
Wind speed (km/h)
Weather condition icon

If the city is not found, an error message is shown.

🔑 API
This project uses the OpenWeatherMap API:
https://openweathermap.org/api
